# 📈 Limit-Up Sniper (涨停狙击手) - 商业版

**Limit-Up Sniper Commercial Edition** 是一款专为 A 股短线投资者打造的 AI 智能辅助决策系统。它深度融合了 **Deepseek 大模型** 的语义分析能力、**实时龙虎榜主力动向** 以及 **新浪财经 L1 实时行情**，致力于在瞬息万变的市场中，为专业投资者提供快人一步的决策支持。

## 🚀 快速开始 (仅限授权用户)

### 1. Windows 用户 (推荐)

我们提供了全自动化的桌面部署版本，路径位于 `Desktop-Version/`。

1.  **安装**: 运行 `install.bat`。自动配置 Python 虚拟环境。
2.  **配置**: 在提示时输入您的 **Deepseek API Key**。
3.  **启动**: 运行 `run.bat`，系统将自动打开浏览器并进入控制台。

### 2. 服务器部署 (专业用户)

如果需要在 Linux 云服务器运行，请参考 `Server-Version/` 下的部署手册。

## 📖 实战指南

1.  **开盘/收盘 深度复盘**
    *   点击 **"开始复盘"**，系统将调用 **Deepseek-V3** 深度解析财联社等主流财经媒体电报。
    *   AI 会自动甄别“竞价抢筹”与“盘中打板”标的，并实时计算买入胜率评分，作为操作参考。

2.  **全自动盘中监控 (09:30 - 15:00)**
    *   系统进入“实战模式”，行情每秒高频刷新，实时监控全市场异动。
    *   **左侧：热门竞价区**。聚焦于集合竞价表现超预期的个股。
    *   **右侧：打板监测区**。动态捕捉大单封板、瞬间拉升等极端异动。

3.  **主力动向研判**
    *   结合个股龙虎榜席位数据（本版特有功能），识别标的是否有知名游资介入，避开散户博弈陷阱。

4.  **快捷仓位管理**
    *   通过右上角搜索框快速建立“私人观察池”，所有标的均支持 AI 深度复盘分析。
